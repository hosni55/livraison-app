# DeliverySupervisionApp

A professional real-time delivery supervision and management system.

## 🚀 Overview
The **Supervision Livraison** application is a comprehensive platform designed to streamline logistics operations. It provides dedicated interfaces for both supervisors (Contrôleurs) and drivers (Livreurs).

## ✨ Key Features

### 👨‍✈️ Supervisor Dashboard (Contrôleurs)
- **Live Analytics**: Real-time charts showing delivery performance per driver and overall success rates.
- **Search & Filter**: Advanced lookup by driver name, client name, or date range.
- **Communication**: Broadcast important information and updates to all drivers on duty.

### 🚚 Driver Portal (Livreurs)
- **Navigation**: One-tap access to Google Maps for precise delivery locations.
- **Status Management**: Real-time status updates with reason-logging for failed deliveries.
- **Emergency Alerts**: Instant emergency messaging to supervisors including order and client context.

## 🛠️ Technology Stack
- **Android**: Native Java with Material Design 3.
- **Backend**: Spring Boot, JPA, Rest API.
- **Database**: MySQL / PostgreSQL (Oracle compatible).
- **Communication**: WebSocket for real-time messaging.
- **Libraries**: MPAndroidChart (DataViz), Retrofit (Networking), Glide (Images).
